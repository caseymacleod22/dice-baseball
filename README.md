## Welcome to Dice Baseball!

## 💻 Technologies Used
- JavaScript
- HTML
- CSS
- Netlify (deployment)

## ⚙️ Features
- Simulates the result of a play with the press of a button.
- Tracks how many outs there are and what inning it is.
- Provides a brief play-by-play for each result.
